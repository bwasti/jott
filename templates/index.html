Welcome to jott.live


Use this site to make simple notes (or TeXdown notes). {{count}} notes made so far.

Command line usage

$ curl https://jott.live/raw/note/note_script > jott && chmod +x jott
$ echo "note contents" | jott <name> <password>
$ jott <name>
note contents

Web

  • Find a saved note at https://jott.live/note/<name>
  • For code syntax highlighting https://jott.live/code/<name>
  • For TeXdown rendering https://jott.live/texdown/<name>
